# Machine Learning Project 1

## Overview
This project demonstrates the implementation of a Machine Learning model using Python. The repository contains the dataset, Jupyter Notebook, and project report required to understand and reproduce the results.

## Repository Structure

```
Machine_Learning/
└── Project_1/
    ├── Python_code.ipynb    # Jupyter Notebook containing the complete implementation
    ├── dataset.csv          # Dataset used for training and testing
    └── Report.docx          # Project report and documentation
```

## Features

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature selection
- Machine Learning model training
- Model evaluation
- Performance visualization

## Technologies Used

- Python 3.x
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/Machine_Learning.git
```

Navigate to the project folder:

```bash
cd Machine_Learning/Project_1
```

Install the required packages:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn jupyter
```

## Usage

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open:

```
Python_code.ipynb
```

Run all the notebook cells sequentially.

## Dataset

The dataset used in this project is:

```
dataset.csv
```

Ensure that the dataset is located in the same directory as the notebook.
